1. npm init -y (Untuk membuat package.json)
2. npm install express mysql2 sequelize cors (install express mysql sequelize dan cors)
3. npm install --save-dev nodemon (install nodemon)
4. tambahkan script "type": "module", di package.json
5. untuk memastikan backend sudah jalan ketikan 'nodemon index di terminal'

Jika tidak ada error maka backend sudah berhasil dibuat
